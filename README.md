# 🌾 AI·드론 기반 청년 농업 창업 입지 진단 시스템 MVP
> **Balanced Regional Development & Youth Agriculture Settlement Solution**  
> 공공 농지·토양·지질 데이터 필터링(1차)과 드론 RGB/열화상 이미지 공간 격자 정밀 분석(2차)을 융합한 청년 창농 입지 진단 의사결정 시스템입니다.

---

## 📌 1. 프로젝트 개요 & 문제 정의

### 1.1 배경 및 문제 정의
대한민국의 **지방 소멸**과 **청년 인구 유출**은 국가적 균형 발전의 핵심적 저해 요소입니다. 정부와 지자체는 청년농 유입 정책(자금, 기술 등)을 활발히 지원하고 있으나, 창창한 포부를 가진 청년 농부들이 정착에 실패하는 가장 큰 원인은 다음과 같습니다:
- **정보 비대칭성**: 농지를 찾고 계약하려 해도 기상, 농업 용수, 경사도, 농기계 진입 도로 등의 필수 정보가 흩어져 있어 파악이 어렵습니다.
- **창농 실패 위험**: 무턱대고 스마트팜 시설을 지었다가 배수 불량, 일사량 부족, 토성 부적합으로 작물 작황에 치명적인 실패를 겪고 파산하는 경우가 빈번합니다.
- **현장 정보 분석 한계**: 위성 사진만으로는 현재 농지가 방치된 채 무성하게 잡초가 자랐는지, 침수 웅덩이가 자주 생기는지 등의 실시간/미시적 특징을 검증할 수 없습니다.

### 1.2 해결 아이디어
본 프로젝트는 **"공공 빅데이터를 통한 광역 농지 스크리닝(1차) + 드론 정밀 이미지를 통한 현장 검증(2차) + AI 작물 및 정책 연계(최종)"**의 3단계 입지 진단 파이프라인을 구축하여 청년 창농의 진입 장벽과 리스크를 혁신적으로 낮춥니다.

## 🚀 2. 핵심 기능 (Core Features)

1. **1차 농지 후보지 선별 (공공 데이터 통합)**
   - 토양 적성 점수, 농업진흥구역 여부, 지목, 도로 접근성, 배수/경사, 지오빅데이터(지하수/지열), 청년 가산점 데이터를 조합해 1차 가중치 점수를 산출합니다.
   
2. **2차 미시 드론 영상 분석**
   - **드론 RGB 분석**: 지상 식생 비율(NDVI 간이 탐색), 나지 비율, 에지 밀도를 통한 농지 정리 상태(균일도), 시설물 설치 공간의 편평도를 자동으로 계산합니다.
   - **드론 열화상(Thermal) 분석**: 표면 밝기 강도를 기반으로 저온 구역(수분 과다/과습/침수 의심)과 고온 구역(가뭄/수분 스트레스)을 감지하고, 위험 영역을 오버레이 표시합니다.
   
3. **추천 작물 및 비즈니스 모델 매핑**
   - 수분 밸런스와 시설 설치 용이성이 높으면 **'스마트팜 딸기/방울토마토'**, 접근성이 높고 소규모면 **'로컬푸드 체험농장'** 등 지목과 환경 조건별 최적 작물을 추천합니다.
   
4. **맞춤형 정부/지자체 청년 지원 정책 추천**
   - 청년 창업 영농정착 지원금, 농지은행 임대형 스마트팜 사업 연계, 지자체 융자 지원 사업 등을 매칭하여 제공합니다.

5. **JSON 진단 리포트 다운로드**
   - 분석 결과 데이터 일체를 표준 JSON 리포트로 즉시 로컬에 저장하고 다운로드할 수 있어, 투자 유치서나 귀농 신청 시 기초 자료로 활용할 수 있습니다.

## 💻 4. 실행 방법 (Usage)

### 4.1 가상환경 생성 및 패키지 설치
Python 3.8 이상 환경에서 터미널 종류(PowerShell 또는 CMD)에 맞춰 가상환경(`venv`)을 활성화하고 패키지를 설치합니다.

#### PowerShell (윈도우 추천)
```powershell
# 1. 가상환경 생성
python -m venv .venv

# 2. 실행 정책 변경 (스크립트 실행 에러 방지) 및 활성화
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
.venv\Scripts\Activate.ps1

# 3. 필수 패키지 설치
pip install -r requirements.txt
```

#### CMD (명령 프롬프트)
```cmd
# 1. 가상환경 생성
python -m venv .venv

# 2. 가상환경 활성화
.venv\Scripts\activate.bat

# 3. 필수 패키지 설치
pip install -r requirements.txt
```

### 4.2 시스템 구동
다음 명령어로 Streamlit 로컬 웹 서버를 실행합니다.
```bash
streamlit run app.py
```
- 브라우저가 자동으로 열리며, 로컬 주소 `http://localhost:8501`로 진입합니다.

## 🌐 5. 향후 확장 방향 및 연동 가능 공공 API 후보

본 MVP는 시연 편의를 위해 Mock 모드로 동작하지만, 향후 상용화를 위해 아래의 표준 오픈 API를 통해 자동으로 데이터를 연동하도록 확장할 예정입니다.

1. **VWorld 지도/공간정보 API (국토교통부)**
   - 연속지적도(LT_C_UQ111), 용도지역(LT_C_UQ111) 등 지번(PNU코드)별 정확한 지목, 공시지가, 경계 기하 정보를 동적 수집.
2. **농업진흥지역도 API (농림축산식품부)**
   - 해당 농지가 행위 제한이 있는 '농업진흥지역(절대농지)'인지 '농업보호구역'인지 공간 질의로 자동 필터링.
3. **팜맵(FarmMap) 공간정보 (농업정보포털)**
   - 위성으로 실측된 경작지 실경계 정보와 지적도 불일치 영역을 감지하여 비정형 경작지 식별.
4. **흙토람 토양 환경 및 작물 적성 데이터 (농촌진흥청)**
   - 토양 배수 등급(양호, 매우양호 등), 유효 토심, 경사도 및 선정된 작물의 토양적성도 점수 Open API 실시간 호출.
5. **지오빅데이터 지하수/지질 정보 (한국지질자원연구원)**
   - 인근 지역의 지하수 일일 취수량, 심도별 지열 에너지 포텐셜, 지반 안정성을 조회하여 대규모 유리온실 설치 적격 여부 판단.
6. **청년농부 지원사업 정보 API (귀농귀촌종합센터 / AGRIX)**
   - 지자체별 영농정착금 잔여 예산 정보, 임대 농지 매칭 혜택 실시간 스크래핑 및 매칭.
